
# Stückliste Compare App

Web estática publicada en GitHub Pages para comparar información de múltiples Excels (uno por modelo de vehículo).

## Características
- Lectura de Excels desde SharePoint (solo lectura)
- Filtros por columnas
- Multiusuario
- Sin backend

## Uso
1. Configura las URLs de SharePoint en `config.js`
2. Sube el repositorio a GitHub
3. Activa GitHub Pages
